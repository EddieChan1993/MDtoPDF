# GrapePress — Claude Context

macOS SwiftUI app (Swift 6, macOS 13+). Converts a folder of Markdown files into a paginated A4 PDF ebook. No external Swift dependencies — pure AppKit/WebKit/PDFKit.

## Architecture

```
AppViewModel  →  MarkdownParser  →  PDFExporter
(orchestrate)    (MD → HTML)        (WKWebView → printOperation → PDFKit outline)
```

**Conversion pipeline:**
1. `AppViewModel.convert()` iterates `.md` files, calls `MarkdownParser.parse()` per file
2. `MarkdownParser.buildPage(chapters:)` assembles full HTML (TOC page + chapter sections)
3. `PDFExporter.export()` loads HTML into an off-screen WKWebView, waits for `didFinish`, then calls `printOperation.runModal(for:delegate:didRun:)` (async — never `op.run()` which blocks the main thread)
4. After print completes, `PDFKit` adds a bookmark outline via `addPDFOutline()`

## Key Files

| File | Role |
|------|------|
| `AppViewModel.swift` | `@MainActor` ObservableObject; drop handling, file list, conversion orchestration |
| `MarkdownParser.swift` | Line-by-line parser; `parse()` → `(html, headings)`; `buildPage(chapters:)` → full HTML |
| `PDFExporter.swift` | Off-screen render + async `runModal` print + PDFKit bookmark patch |
| `ContentView.swift` | Two-state SwiftUI UI: drop zone (empty) → file list (loaded) |
| `make_icon.py` | Generates flat SVG → all icon sizes → `AppIcon.icns` via rsvg-convert + iconutil |
| `build.sh` | SPM compile + `.app` bundle assembly + icon copy + ad-hoc codesign |

## Critical Invariants

- **Never use `op.run()`** for print operations — it runs a synchronous modal loop that blocks the main thread and freezes the UI. Always use `op.runModal(for:delegate:didRun:contextInfo:)`.
- **WKWebView must be in an ordered-in window** (`orderFrontRegardless()` + `alphaValue = 0`) to trigger `didFinish navigation`. `orderBack(nil)` on a never-shown window doesn't work.
- **NSItemProvider is not Sendable.** Call `loadItem` on the main thread; use `Task { @MainActor in }` only for the result callback. Don't pass the provider into a Task or nonisolated context.
- **`PDFDocument.index(for:)` returns `Int` (not `Int?`)** on macOS — don't use `guard let` or optional binding on it.
- Bookmark search starts at page index 1 (`searchStart = 1`) to skip the TOC page (index 0).
- Images use `.removingPercentEncoding` + `.standardized` to handle URL-encoded paths and `../../` traversal.

## Markdown Parser Details

- `parse(_:baseDirectory:)` → `(html: String, headings: [Heading])`
- `buildPage(chapters:)` → full HTML with A4 CSS, TOC, and chapter sections
- TOC lists chapter filenames only (no sub-headings)
- Chapter sections: `page-break-before: always` except first
- Images: decoded path → standardized URL → NSImage → resize ≤1000px → JPEG 0.72 → base64 data URI
- Blockquote `>`: empty lines silently skipped; `> ## heading` parsed recursively; `> text` → `<blockquote>`
- Heading IDs generated by `makeHeadingID()` with duplicate counter
